# Task Tracker CLI

A simple command-line TODO/task tracker built with Python and functional programming principles.

## Features

- ✅ Add tasks
- 📋 List tasks
- 🗑️ Delete tasks
- 💾 Stores tasks in JSON file

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
